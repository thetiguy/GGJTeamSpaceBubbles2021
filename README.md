# GGJTeamSpaceBubbles2021
## 2021 Theme
Lost and Found
## Name Suggestions
Mystery Gang
## How To Play
Drag investigtors onto the locations to assign them to work.
Click the clues in the file list to view each clue as they come in
You need to tell Ren how many, of which symbol to push, in which order
e.g. 6 suns 6 spirals 6 stars
